## FastMending

Плагін дозволяє миттєво одноразово ремонтувати предмети через ковадло за допомогою книги з зачаруванням **Лагодження**. Зачарування не залишається на предметі, лишень ремонтує повністю предмет за кількість рівнів, вказаних в конфігурації - `config.yml`. 

### Як це працює

Покладіть пошкоджений предмет у ліву комірку ковадла, а книгу з **Лагодження** — у праву. Предмет буде відремонтовано до повного стану.  
_**НЕ** пошкоджений предмет, що логічно, не можна ремонтувати._

### Команди

| Команда     | Опис                                 |
|-------------|--------------------------------------|
| `/fmreload` | Перезавантажити конфігурацію плагіна |

### Права

| Право                | Опис                                 |
|----------------------|--------------------------------------|
| `fastmending.reload` | Дозволяє використовувати `/fmreload` |

### Конфігурація

```yaml
enabled: true        # Увімкнути/вимкнути плагін
repair-cost: 1       # Вартість ремонту в рівнях досвіду

messages:
  reload-success: "&aКонфігурацію FastMending успішно перезавантажено!"
  no-permission: "&cУ вас немає прав на використання цієї команди!"
```

> Для кольорів у повідомленнях використовуйте `&` з кодом кольору, наприклад `&a` — зелений, `&c` — червоний.
